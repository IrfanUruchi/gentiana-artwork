# Gentiana Artwork

Portfolio website for my sister, Gentiana Uruçi, an impressionist painter and Master of Painting from North Macedonia.

The website showcases her paintings from 2018–2024 and serves as her personal online gallery and portfolio.

## Built with

- React
- Vite
- React Router
- CSS
- Cloudflare Pages

## Features

- Year-based collections
- Full-screen image viewer
- English and German support
- Responsive design
- Optimized images

## Development

```bash
npm install
npm run dev
npm run build
```
